Reflections on CIS 241

Technology changes constantly and quickly. That is the number one thing I can take away from this class. While the authors of the class are clearly trying to provide valuable knowledge and advice to their students, portions of the content are already out of date. HTML 5 standards and adoption have moved quickly. Things that were true about the web when the material was written have already passed into obsolescence. The core of HTML remains solid and valid, but the ways in which it is used are ever evolving. Anyone who wants to be successful as a web developer needs to be always learning.


The fundamental aspects of web browser and web server, HTML as a set of markup tags, and even a lot of the actual tags remain much as they were when I first learned about them in the 1990s, but there is now an entire universe built up around web development. Bootstrap and Semantic UI, frameworks like React and Angular, nodejs and the hundreds of freely available node modules, there is no end. It may be cliche, but a person could spend a lifetime studying and never learn it all.
